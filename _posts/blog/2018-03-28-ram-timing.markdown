# RAM的速度
DDR1，DDR2，DDR3内存是按照工作时能达到的最大速度和时序来分类的。时序是一串数字，比如，
3-4-4-8, 5-5-5-15, 7-7-7-21, 9-9-9-24等，数字越小越好。在本教程中，解释这些数字的意义。

DDR1，DDR2，DDR3内存符合 DDRxxx/PCyyyy分类法。
